## Getting started
```  
git clone https://github.com/shreeyansh-29/Mini-Page-Builder
cd Mini-Page-Builder
npm install  
```  
Running the below command will start the application  
```  
npm start  
#(Local application will run on port 3000)  
```   
  
## Prerequisites
1.IDE (visual studio)  

2.npm version 6.14.11

3.react v18.2.0

4.node v14.15.5
